# 백종원 레시피 추천 모바일 앱

# 개요
- 백종원 레시피들만 모여있는 모바일 앱
- 웹 크롤링의 첫 사용


# 주요 기능
- 현재 가지고 있는 재료를 입력하여 만들 수 있는 음식들의 레시피들만 골라서 추천 받을 수 있는 기능
- DB에 있는 모든 음식 레시피들 중 키워드로 레시피 검색이 가능


# 사용된 Stack
- 앱 : Android Studio Java
- DB : Firebase, SQLite


# 내가 사용한 Stack
- Java
